## 2. やったこと
ハッカソンの時間は限られてますので、まずは「とにかく動くもの」を目標にしました。  
そこで、Dockerコンテナ上で **gemini CLI** を使い簡単なエージェントを作ることに挑戦しました。
実施内容は以下3点です。

- 2.1 やりたいことを考える
- 2.2 セットアップする
- 2.3 エージェントを作ってみる

### 2.1. やりたいことを考える
最初に決めたテーマは **「やりたいことを教えてくれるエージェント」** にしました。
そもそも、AI活用にあたりに「何をやるのか？」を考えるのが大事で、テーマを悩みそうになったのでそれ自体をテーマとしました笑。

### 2.2. セットアップ
方針が決まったら環境構築です。
無料でサクッと動かしたかったのでgemini CLIを選び、ローカル環境への影響を避けるためにDocker上で実行します。

1. Dockerコンテナを立ち上げる  
2. gemini CLI をセットアップ  
3. 簡単に対話して動作確認  

手順はこちらの記事を参考にします。
https://zenn.dev/yulog/scraps/80b200ff6fec08

無事、応答が返ってくるところまで確認できました。  

![](/images/18_event_report_ai_agent/1_gemini.png)


## 2.3. エージェント作成
次にgemini CLIに相談をなげてみます。

:::details 相談内容
```bash
╭──────────────────────────────────────────────────────────────────────────────╮
│  > やりたいことを教えてくれる教えてくれるエージェントを作ろうと考えてます。  │
╰──────────────────────────────────────────────────────────────────────────────╯

✦ 「やりたいことを見つけてくれるエージェント」、面白いアイデアですね！素晴らしいと思います。

  そのエージェントは、どのようにして「やりたいこと」を提案するのでしょう？
  例えば、ユーザーにいくつか質問をして、その答えから興味や関心を分析するのでしょうか？
  それとも、ユーザーの普段の行動から学習するのでしょうか？

  どのような形（Webアプリ、スマホアプリ、CLIツールなど）で実現したいか、具体的なイメージはありますか？

  ぜひ、そのアイデアの実現をお手伝いさせてください。

  手始めに、この新しいエージェントのためのプロジェクトディレクトリを作成し、基本的な設計を考えるところから始めては
  いかがでしょうか。

╭──────────────────────────────────────────────────────────────────────────────────────────────────────────────────╮
│  > 対話形式で考えるようなイメージです。今回docekrコンテナないにたててますがアプリみたいにコード作ってもらっても │
│    いいですか？                                                                                                 │
╰──────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯

✦ 承知いたしました。対話形式のアプリケーション、素晴らしいですね。早速コードを作成しましょう。

  まずは、シンプルで分かりやすいPythonを使ったコマンドラインアプリケーションとして作成するのはいかがでしょうか。

  このアプリは、以下のような対話を行います。

   1. アプリがユーザーに気分やエネルギーレベルについて簡単な質問をします。（例：「今の気分は？」「体を動かしたい
      気分？」）
   2. ユーザーの回答に基づいて、おすすめのアクティビティを一つ提案します。（例：「それなら、近所を散歩して新しい
      カフェを探すのはどうでしょう？」）

  このような形で進めてもよろしいでしょうか？

```
:::

## 3. 感想

会場では、gemini cliをColab上で動かしている人や、MCPサーバを立てている人もいて、ハッカソンらしいな〜と感じました。

そして、甲子園決勝と重なったこともあり、決勝をみんなでみようとイベント開始時間を決勝時間に変更してるのがとても沖縄らしくてよかったです笑。
